Sol's Stat Tracker Webhook is a WebSocket client that connects to Sol's Stat Tracker API and automatically posts game statistics to your Discord server through webhooks. Get real-time notifications whenever stats are tracked, with customizable embed colors, emojis, and webhook appearance.

## Key features

**Real-time WebSocket connection**
Connects to Sol's Stat Tracker API via WebSocket for instant stat delivery to your Discord server.

**Automatic reconnection handling**
Built-in exponential backoff reconnection logic ensures your webhook stays connected even during temporary outages.

**Customizable webhook appearance**
Override the webhook username and avatar URL to match your server's branding and style.

**Configurable embed styling**
Customize embed colors and emojis for success, error, and neutral states to match your preferences.

**Verbose logging options**
Toggle automated event logging to control how much detail you see in your Discord channel.

**Token-based authentication**
Secure API token authentication generated directly through the Sol's Stat Tracker Discord bot.
